# **在 Jetson Nano 上使用 YOLOv4 進行口罩辨識**  
# **簡介**
## **為何選用 YOLOv4 **  
1.設備需求低  
2.運算速度相比R-CNN更快  
3.正確率高  
## **為何選用 Jetson Nano**  
1.耗電量低  
2.運算能力高  
3.體積小
